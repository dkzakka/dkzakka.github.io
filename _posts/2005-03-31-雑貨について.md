## 〈 ECONOMY LETTER 〉
梱包後の厚みが 3cmまでに限られています

```
A　340円（〜20g）
B　550円（〜50g）
C　800円（〜100g）
D　1380円（〜250g）
E　2410円（〜500g）
F　3450円（〜1000g）
G　6750円（〜2000g）
```

追跡番号をお付けする場合は

お知らせくださいませ

## 〈 MAXI LETTER 〉
郵便追跡サービスをご利用の場合はお知らせ下さいませ

```
H 2760円（〜250g）
I　4830円（〜500g）
J 7500円（〜1000g）
K 9350円（〜1500g）
L 13120円（〜2000g）
```

## 〈　国際郵便扱い　〉

トラッキングナンバー付きです

梱包後のサイズによって料金がかわります。

目安としては以下の通りになります。

```
S　10kg以下　サイズ11×32×42　6900円
M　10kg以下　サイズ19×36×60 11250円
L　10kg以下　サイズ36×37×60 18000円
XL 10kg以下　サイズ36×59×60 25500円
```
